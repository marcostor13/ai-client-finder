# AI Agent Configuration & Guidelines

## Optimization (Token Efficient)
- Think before acting. Read existing files before writing code.
- Be concise in output but thorough in reasoning.
- Prefer editing over rewriting whole files.
- No sycophantic openers or closing fluff (e.g., "Sure!", "I hope this helps!").
- Keep solutions simple and direct.
- User instructions always override this file.

## UI/UX Design System (Skills)
- **Aesthetic**: Premium, state-of-the-art, modern and vibrant.
- **Color Palette**: 
  - Primary: Vibrant Purple (`#6D28D9`)
  - Secondary: Deep Indigo (`#4C1D95`)
  - Accents: Neon Magenta or Cyan for highlights.
  - Background: Sleek Dark Mode (Deep Greys/Blacks) or ultra-clean glassmorphism.
- **Styling Rules**:
  - Use Vanilla CSS or CSS Modules. **No TailwindCSS** unless explicitly requested.
  - Modern Typography: Use 'Inter' or 'Outfit' from Google Fonts.
  - UI Elements: Smooth gradients, rounded corners (large radius), subtle box shadows.
  - Effects: Glassmorphism (`backdrop-filter: blur()`), micro-animations on hover/click.
  - Dynamic: The interface must feel alive and responsive.

## Project Context
- **Backend**: Python (FastAPI), MongoDB, JWT Auth.
- **Frontend**: React (Vite).
- **Goal**: AI Client Finder using Agents and Web Scraping.
