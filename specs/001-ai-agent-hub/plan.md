# Implementation Plan: AI Agent Hub

**Branch**: `001-ai-agent-hub` | **Date**: 2026-06-27 | **Spec**: [spec.md](./spec.md)

## Summary

Build a new "AI Agent Hub" page (`/agent`) and backend module (`backend/agent_hub/`) that provides a conversational AI assistant backed by a resilient multi-model gateway. The gateway detects the intent of each message and routes to the best available free-tier model (text, image, audio, TTS), with automatic fallback when a model fails or exhausts its daily quota. A Connections panel lets users link Outlook Calendar (Microsoft Graph OAuth2), Telegram (bot webhook), and WhatsApp (WAHA QR-code session) so the agent can interact across all three channels.

## Technical Context

**Language/Version**: Python 3.13 (backend) · Node 20 / React 19 (frontend)

**Primary Dependencies**:
- Backend new: `msal` (MS Graph OAuth2), `python-telegram-bot==21.*` (Telegram webhook), `cryptography` (token encryption at rest), `httpx` (async HTTP — already in project via other deps)
- Frontend: React 19, React Router 7, Axios, Lucide React — all already installed

**Storage**: MongoDB Atlas via Motor (existing); five new collections listed in data-model.md

**Testing**: Manual validation via quickstart.md (project has no automated test suite)

**Target Platform**: Linux server (Coolify/Docker) + Netlify CDN (frontend)

**Project Type**: Full-stack web service — follows existing backend/frontend split

**Performance Goals**: Text responses ≤ 10 s P90; Telegram/WhatsApp replies ≤ 15 s; QR code displayed ≤ 5 s after session create

**Constraints**:
- Single-worker FastAPI process (APScheduler constraint — cannot change)
- WAHA Core edition (free, open source) — unlimited sessions, no message limits
- Refresh tokens for Microsoft Graph with `offline_access` last 90 days by default; longer if tenant is configured for sliding window
- MS OAuth app registration is operator-managed (not per-user)
- Free model quotas are the primary throughput ceiling

**Scale/Scope**: Single-user to small-team; not designed for high-volume public traffic
